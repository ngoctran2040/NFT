Superex - React Multipurpose Template

Introduction

Superex is a Powerful React NFTs marketplace template. Superex is a fully responsive, pixel-perfect, and the most complete template NFTs, crypto-collectibles, digital items, non-fungible tokens websites, MetaMask Integration, and more. Superex is function components based react template. Superex has a lot of styles and elements to help you make beauty and modern website in no time. Superex offers various personalization options to help you give the look you require on your website. It is very easy to Customization, navigate, and fits various screen sizes.

Superex provides fully responsive a choice to choose LTR or RTL version supported in both Light and Dark mode managed with CSS based.

How to change the fonts family in Superex?

If you want to change your font family then Please first of all open the _variables.scss (path: src/assets/scss/_variables.scss) and add your font family @import url('https://fonts.googleapis.com/css2?family=Poppins:wght@400;500;600;700&family=Roboto&display=swap'); Import the @import link from your Google Fonts.

And also add or change to your font family name in,
$font-family-base: 'Poppins', sans-serif;
$font-family-secondary: 'Roboto', sans-serif;

How to change the Template or Theme Mode (Light or Dark) in Superex?

If you change the theme or template mode Light to Dark please follow the below step, Please first of all open the App.js (path: src/App.js) find the import "./assets/scss/style.scss" and replace or changes with import "./assets/scss/style-dark.scss"

Note:
Light Mode: import "./assets/scss/style.scss"
Dark Mode: import "./assets/scss/style-dark.scss"

How to change the Template or Theme Mode (LTR or RTL) in Superex?

If you change the theme or template mode LTR to RTL please follow the below step, Please first of all open the App.js (path: src/App.js) find the import "./assets/scss/style.scss" and replace or changes with import "./assets/scss/style-rtl.scss"

Note:
LTR Mode: import "./assets/scss/style.scss"
RTL Mode: import "./assets/scss/style-rtl.scss"
Dark RTL Mode: import "./assets/scss/style-dark-rtl.scss"

How to change the Theme primary color in Superex?

If you change the theme or template color please follow the below step, Please first of all open the _variables.js (path: src/assets/scss/_variables.scss) find the $primary and replace or changes with color code

like:
$primary: #2a52c3; replace only #2a52c3 to your color (#f89d36)
